## 🎯 기능 구현 목록
- RacingCarGame start
- Car 관련 모든 데이터 통제를 이루는 Service
- UserInput 으로부터 이름 전달 받아서 Car 생성 후 CarRepository에 등록
- 반복횟수(Lap) 입력 및 Lap 만큼 게임 반복
- 게임진행 내용 및 결과 출력
- 우승자(WinnerRepository)에 우승자 등록
- 우승자 출력

<br>

## ☑️ 예외처리 목록
### Car 이름
- [x] 이름 길이 5자 이하
- [x] 이름 사이 공백 포함
- [x] 이름길이 0 (공백)
- [x] 이름 중복
- [x] 이름목록이 쉼표(,)로 끝남

### 반복 횟수(Lap)
- [x] 정수가 아님
- [x] 음수 입력
- [x] 처리할 수 없는 너무 큰 수 입력
